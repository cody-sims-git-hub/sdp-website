# Sims Digital Partners Website

Personal portfolio and professional website for Cody Sims.

**Website:** https://simsdigitalpartners.com

## Built with

- React
- TypeScript
- Vite
- Tailwind CSS

## Purpose

- Professional portfolio
- Project showcase
- Contact point for software, application, and automation opportunities

## Featured projects

- **Sentinel Vision** — real-time object detection (YOLOv8m, PyTorch, CUDA, OpenCV)
- **SimpleCRM** — Laravel customer relationship management app

## Development

```bash
npm install      # install dependencies
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # production build (static output in build/client/)
npm run typecheck
```

The site is statically prerendered; deployment is the contents of `build/client/`.
